# Let Me Ask Claude

For when someone asks you something they could just ask Claude.

Generate a shareable link with a question. When opened, the link animates typing the question into a Claude-like UI, then redirects to [claude.ai](https://claude.ai) with the question pre-filled.

Live at [letmeaskclaude.aallam.com](https://letmeaskclaude.aallam.com)

## Development

```sh
npm install
npm run dev
```

## Build

```sh
npm run build
```
